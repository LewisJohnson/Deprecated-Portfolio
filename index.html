Projects.

  • Cycle London

    C# - XAML - Windows 8.X Universal

    Cycle London is 2 projects. The Windows 8 app was developed first, then succeeded by the Windows 10 UWP app, London Universal. The app uses mapping & TFL (Transport For London) API's to help Londoners navigate London by bike with ease. Please visit the repo on GitHub.

    Cycle London Repository

    Cycle London Demonstration (YouTube)

  • London Universal

    C# - XAML - Windows 10 UWP

    London Universal is an app, which helps Londoners navigate the city by showing travel information for everyone, no matter how they're getting around. Cycle Hire, Super Cycle Highways, CabWise, Tube Stations, Oyster TopUp Spots & much more.

    London Universal Repository

    London Universal Demonstration (YouTube)

  • SketcherBook Pro

    C# - XAML - Windows 10 UWP

    SketcherBook Pro is a full featured drawing app for Windows 10. The app includes a bunch of great features including editable layers, sharing your drawings with on social media, saving your drawing as an image or to edit later, Graphics tablet support and much more.

    SketcherBook Pro Repository

  • p2crypt

    C# - XAML - Windows WPF

    p2crypt was an open source, peer-to-peer, encrypted chat client for Windows. The app looks great and is a lightweight alternative to other applications. Unfortunately, the servers have since been terminated.
